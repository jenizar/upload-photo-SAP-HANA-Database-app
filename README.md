# upload-photo-SAP-HANA-Database-app
Upload photo SAP HANA Database application using python streamlit

pip install hdbcli streamlit

run application:

$>streamlit run app.y

Local URL: http://localhost:8501

Network URL: http://192.168.0.12:8501
